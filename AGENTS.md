# Codex Project Guidance

- Follow `sluice-ui-brand-rules.md` for visual and implementation direction.
- Use `Instrument Serif` for display headings and large editorial numbers.
- Use `Inter` for body, UI, navigation, labels, captions, and buttons.
- Do not replace the approved `Instrument Serif` and `Inter` stack unless the user explicitly asks.
- Keep the hero route diagram responsive without page-level horizontal scroll. Oversized desktop art may be clipped by the viewport, but it must not create a sideways scrollbar.
- Do not use CSS transforms as the primary way to size the hero diagram.
- On mobile, keep the hero order as title, route diagram, then CTA buttons. CTA text must remain on one line.
